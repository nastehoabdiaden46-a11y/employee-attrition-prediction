# 👥 Employee Attrition Prediction

A full-stack Machine Learning application that predicts whether an employee is likely to **stay** or **leave** an organization. The project combines a trained Machine Learning model, a FastAPI backend, and a modern Next.js frontend to provide real-time employee attrition predictions.

## 🚀 Live Demo

- **Frontend:** https://employee-attrition-prediction-k8ee.vercel.app
- **Backend API:** https://employee-attrition-prediction-qroi.onrender.com
- **API Documentation:** https://employee-attrition-prediction-qroi.onrender.com/docs

---

## 📌 Project Overview

Employee attrition is a major challenge for organizations because losing experienced employees increases recruitment costs, reduces productivity, and impacts business performance. This project was developed to help HR teams identify employees who may be at risk of leaving by using Machine Learning.

The application allows users to enter employee information through an intuitive web interface. The data is sent to a FastAPI backend where a trained Machine Learning model analyzes the features and predicts whether the employee is likely to stay or leave. In addition to the prediction, the system provides:

- Employee attrition prediction
- Stay probability
- Attrition probability
- Risk level (Low, Medium, High)
- Model information
- Real-time API responses

The backend is built with **FastAPI** and serves the trained model through REST APIs, while the frontend is built with **Next.js**, **TypeScript**, and **Tailwind CSS** to deliver a modern, responsive user experience. The application is fully deployed with **Render** for the backend and **Vercel** for the frontend, making it accessible online.
